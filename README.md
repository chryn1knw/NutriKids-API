## Instalasi

```
# Clone repo
git clone https://github.com/namamu/NutriKids-API.git
cd NutriKids-API

# Buat virtual environment
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# Install requirements
pip install -r requirements.txt
```

## Menjalankan API
```
python src/app.py
```

# Jalankan semua tes
```
pytest -v
```
